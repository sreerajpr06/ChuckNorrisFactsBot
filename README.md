## Setting up 

### In Windows systems
Installation of virtual environment 
    python -m venv env

For activating the virtual environment
    .\env\Scripts\activate.bat

For installing all dependencies
    pip install requirements.txt

For running 
    python main.py

For deactivating the virtual environment
    deactivate
